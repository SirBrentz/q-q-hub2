Quidget at a glance

A single source of truth for what Quidget does today, what it doesn't, and what a clinic needs to launch. Use this when scoping a new prospect.

Live with partners · Plugin v1.0.6
01

What Quidget supports — and what it doesn't

Capabilities at the product level
DOs Quidget handles this today
  • Good Faith Exams (GFEs)
  • Single-Rx checkout
  • Instant video consultation post-checkout
  • Linking to pre-existing products
DON'Ts Not supported today
  • Multi-Rx
  • Multiple locations (franchises)
  • Payment splitting
02

Plugin limitations

Constraints to flag during scoping

Multi-Rx not supported

The plugin handles single-Rx flows only. Clinics with bundled or stacked prescriptions can't run that flow through Quidget today.

Roadmap

Plugin compatibility

A known list of WordPress plugins conflicts with Quidget. Run a compatibility check on any prospect's site before promising a launch date.

List maintained separately

Custom exams not supported

Only Qualiphy's standard exam catalog runs through Quidget. Clinics asking for bespoke exam logic should be routed off-Quidget for now.

Catalog-only
03

Themes & environment

What the plugin runs on
Supported themes Hello Theme by Elementor Twenty Twenty-Five Other themes may work but aren't actively tested.
PHP version 7.4 or higher Plugin enforces this minimum at install time.
WordPress 5.0 or higher Required core version.
WooCommerce 5.0 or higher · required WooCommerce must be installed and active.
04

Requirements to get started

What a prospect needs in place before onboarding
1
WooCommerce setup
Installed, active, and configured for storefront checkout.
Required
2
WordPress site
Self-hosted or managed WordPress install on their own domain.
Required
3
Qualiphy account with API key access
Active Qualiphy account with API credentials provisioned.
Required
4
SSL certificate
Valid SSL on the domain. The plugin auto-disables on repeated SSL failures.
Required
5
Single location
One clinic location. Multi-location franchise setups aren't supported today.
Required
6
PNG-formatted logos for branding
Provide on a transparent background for white-label branding inside the widget.
Optional
7
Domain compatibility check
Audit the prospect's site against the known incompatible-plugins list, theme, and PHP / WP / WooCommerce versions before greenlighting onboarding.
Pre-launch
