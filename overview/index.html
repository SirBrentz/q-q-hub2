Quidget at a glance

A single source of truth for what Quidget does today, what it doesn't, and what a clinic needs to launch. Use this when scoping a new prospect.

Live with partners · Plugin v1.0.6
01

What Quidget supports — and what it doesn't

Capabilities at the product level
DOs Quidget handles this today
  • Good Faith Exams (GFEs)
  • Single-Rx checkout
  • Instant video consultation post-checkout
  • Linking to pre-existing products
DON'Ts Not supported today
  • Multi-Rx
  • Multiple locations (franchises)
  • Payment splitting
02

Plugin limitations

Constraints to flag during scoping

Multi-Rx not supported

The plugin handles single-Rx flows only. Clinics with bundled or stacked prescriptions can't run that flow through Quidget today.

Roadmap

Plugin compatibility

A known list of WordPress plugins conflicts with Quidget. Run a compatibility check on any prospect's site before promising a launch date.

List maintained separately

Custom exams not supported

Only Qualiphy's standard exam catalog runs through Quidget. Clinics asking for bespoke exam logic should be routed off-Quidget for now.

Catalog-only
03

Requirements to get started

What a prospect needs in place
1
A WordPress site with WooCommerce
Their site needs to be on WordPress with WooCommerce installed and handling their checkout.
Required
2
A Qualiphy account
Active account with API key access — that's how Quidget talks to Qualiphy.
Required
3
A secure site (HTTPS)
Their domain needs a valid SSL certificate. Most modern hosts include this by default.
Required
4
A single clinic location
Quidget is one clinic per install today. Multi-location franchises aren't supported yet.
Required
5
Logo on a transparent background (PNG)
For white-label branding inside the widget. Nice to have, not a blocker.
Optional
6
Run the compatibility scanner below
Drop their domain into the scanner before pitching a launch — it flags conflicts in seconds.
Pre-launch
04

Compatibility scanner

Paste a prospect's domain — get an instant verdict

Drop in any prospect's website and we'll check the things that actually matter: is it a WordPress site, is it running WooCommerce, and is anything on the site likely to conflict with Quidget. Takes 15–30 seconds.
