Two things every prospect needs

Required to use Quidget

A Qualiphy account with API key access
That's how Quidget talks to Qualiphy.
A WooCommerce site
Which means a WordPress site with WooCommerce active for checkout.

At a glance

What Quidget does

  • Good Faith Exams (GFEs)
  • Single-Rx checkout
  • Instant video consultation post-checkout
  • Linking to pre-existing products

What Quidget doesn't do

  • Multi-Rx
  • Multiple locations (franchises)
  • Payment splitting
  • Custom exams — only Qualiphy's standard exam catalog runs through

Other things to know

  • Site needs HTTPS — most modern hosts include this by default
  • Single clinic location only — multi-location franchises aren't supported yet
  • Some WordPress plugins conflict with Quidget — use the floating Scan a site button to check before pitching
